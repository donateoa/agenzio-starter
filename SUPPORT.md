# Support Policy

## Community Support

Agenzio Starter is open source software provided as-is. Support is provided on a best-effort basis through:

- **GitHub Issues** - Bug reports with reproducible steps
- **GitHub Discussions** - Questions and community help

## What We Support

- Installation and setup issues with clear reproduction steps
- Bug reports that include environment details and minimal reproduction
- Documentation improvements and corrections
- Feature requests (implementation not guaranteed)

## What We Do Not Support

- Custom application logic or business requirements
- Third-party integrations not included in the template
- GCP/Firebase billing, quotas, or account issues
- Production incident response or on-call support
- Private consulting or debugging sessions via issues
- Guaranteed response times or SLAs

## Before Opening an Issue

1. Search existing issues to avoid duplicates
2. Include your environment (Node version, OS, Terraform version)
3. Provide minimal reproduction steps
4. Include relevant error messages and logs

## Contributing

We welcome contributions. Before submitting a pull request:

1. Open an issue to discuss the change
2. Keep changes focused and well-scoped
3. Include tests if applicable
4. Update documentation as needed

## No Roadmap Promises

This project does not maintain a public roadmap. Feature requests may or may not be implemented based on maintainer availability and project direction.
